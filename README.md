Prerequisites 
================
* Be connected to the internet.
* MTN github user with github installed.
* ruby 1.9.3 installed.



To Start
================
* run: `sh seanet-demo-bootstrap.sh`
* open INCOGNITO chrome browser window. (CMD+SHIFT+N)
* navigate to http://localhost:4567



To Reset
================
* run: `sh reset.sh`
